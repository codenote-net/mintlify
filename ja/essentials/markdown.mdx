## タイトル

セクションヘッダーに最適です。

```md
## タイトル
```

### サブタイトル

サブセクションヘッダーに最適です。

```md
### サブタイトル
```

<Tip>

各**タイトル**と**サブタイトル**はアンカーを作成し、右側の目次にも表示されます。

</Tip>

## テキストフォーマット

ほとんどのmarkdownフォーマットをサポートしています。テキストの周りに`**`、`_`、または`~`を追加するだけでフォーマットできます。

| スタイル | 書き方 | 結果 |
| ------------- | ----------------- | --------------- |
| 太字 | `**太字**` | **太字** |
| 斜体 | `_斜体_` | _斜体_ |
| 取り消し線 | `~取り消し線~` | ~取り消し線~ |

これらを組み合わせることができます。例えば、`**_太字と斜体_**`と書くと**_太字と斜体_**のテキストが得られます。

上付き文字と下付き文字のテキストを書くにはHTMLを使用する必要があります。つまり、テキストの周りに`<sup>`または`<sub>`を追加します。

| テキストサイズ | 書き方 | 結果 |
| ----------- | ------------------------ | ---------------------- |
| 上付き文字 | `<sup>上付き文字</sup>` | <sup>上付き文字</sup> |
| 下付き文字 | `<sub>下付き文字</sub>` | <sub>下付き文字</sub> |

## ページへのリンク

テキストを`[]()`で囲むことでリンクを追加できます。`[googleへのリンク](https://google.com)`と書くと[googleへのリンク](https://google.com)になります。

ドキュメント内のページへのリンクは、ルート相対である必要があります。基本的に、フォルダパス全体を含める必要があります。例えば、`[テキストへのリンク](/writing-content/text)`は、コンポーネントセクションの「テキスト」ページにリンクします。

`[テキストへのリンク](../text)`のような相対リンクは、最適化しにくいため、開くのが遅くなります。

## ブロッククォート

### 単一行

ブロッククォートを作成するには、段落の前に`>`を追加します。

> ドロシーは彼女に従って城の美しい部屋の多くを通りました。

```md
> ドロシーは彼女に従って城の美しい部屋の多くを通りました。
```

### 複数行

> ドロシーは彼女に従って城の美しい部屋の多くを通りました。
>
> 魔女は彼女に鍋とやかんを洗い、床を掃除し、薪で火を絶やさないよう命じました。

```md
> ドロシーは彼女に従って城の美しい部屋の多くを通りました。
>
> 魔女は彼女に鍋とやかんを洗い、床を掃除し、薪で火を絶やさないよう命じました。
```

### LaTeX

MintlifyはLatexコンポーネントを通じて[LaTeX](https://www.latex-project.org)をサポートします。

<Latex>8 x (vk x H1 - H2) = (0,1)</Latex>

```md
<Latex>8 x (vk x H1 - H2) = (0,1)</Latex>
```
